Tentangsaya saat ini

Mahasiswa, Perguruan tinggi
Universitas Nusantara PGRI Kediri

Saya adalah seorang mahasiswa Teknik Informatika yang saat ini berada di semester 6. Saat ini, saya sedang berfokus dalam bidang keilmuan sebagai pengembang web, saya telah memiliki sedikit ketrampilan dalam membuat web menggunakan Laravel dan pengembangan web secara manual tanpa menggunakan framework (web Vanilla). Keterampilan ini saya dapat ketika saya mengikuti MSIB Kampus Merdeka di PT. NURUL FIKRI dengan materi pembelajaran FULLSTACK WEB DEVELOPER.

Di samping berkuliah, saya juga menjalankan pekerjaan sampingan sebagai tutor di RUANG ROBOT. saya memiliki tanggung jawab sebagai pembimbing dan pengajar dalam berbagai kegiatan ekstrakurikuler dan pembelajaran privat di bidang robotika.
Pekerjaan sebagai tutor di RUANG ROBOT memungkinkan saya untuk menimba ilmu robotika dan memperluas wawasan yang telah saya miliki, serta akan menambah pengalaman bagi saya.
